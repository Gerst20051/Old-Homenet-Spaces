ECHOCONFIG

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays messages during the processing of the MS-DOS subsystem CONFIG.NT and AUTOEXEC.NT when the MS-DOS subsystem is invoked. If omitted, messages will not be displayed.
