NET VIEW

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays a list of resources being shared on a computer.

Syntax

NET HELP VIEW

NET VIEW [\\computer_name [/CACHE] | /DOMAIN[:domain_name]] [/YES]

NET VIEW [/NETWORK:NW] [\\computer_name]

NET VIEW [/WORKGROUP:wgname] [/YES]

To display a list of computers in your workgroup that share resources:

NET VIEW

Parameters
\\computer_name (v7.00 Win95 NT3.1)
Specifies the name of the computer whose shared resourced you want to see listed.
Switches
/CACHE (NT2000 ONLY)
Displays the offline client caching settings for the resources on the specified computer.
/DOMAIN[:domain_name] (NT3.1)
Specifies that you want to view the names of the computers in another domain that share resources.
/NETWORK:NW (NT3.5)
To see a list of shares on a remote Netware computer.
/WORKGROUP:wgname (v7.00 Win95)
Specifies that you want to view the names of the computers in another workgroup that share resources.
/YES (v7.00 Win95)
Carries out the command without first prompting you to provide information or confirm actions.
