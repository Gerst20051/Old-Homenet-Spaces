NLBMGR

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Network Load Balancing Manager can configure and manage Network Load Balancing clusters and all cluster hosts from a single computer, and can also replicate the cluster configuration to other hosts.

Syntax

NLBMGR [/help]

NLBMGR [/autorefresh [interval]] [/hostlist file_name] [/noping]

Parameters

none.

Switches
/autorefresh [interval] (NT2003)
Causes refresh of host and cluster information every interval seconds. If interval omitted, the information is refreshed every 60 seconds.
/help (NT2003)
Display command help.
/hostlist file_name (NT2003)
After starting, the hosts specified in file_name will be loaded.
/noping (NT2003)
Will not ping the hosts prior to trying to contact them through Windows Management Instrumentation (WMI) Use if you have disabled ICMP on all available network adapters. You will experience a delay attempting to contact a host that is not available.
