MRINFO

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Queries a specified multicast router with an Internet Group Management Protocol (IGMP) message. The response to the query contains a version number, the list of interfaces and the neighbors on each interface, metrics, Time to Live (TTL) thresholds, and flags.

Syntax

MRINFO [-i address] [-n] [-r retry_count] [-t timeout_count] multicast_router

Parameters
multicast_router (NT2003)
IP address.
Switches
-i address (NT2003)
Specifies the IP address of the interface from which you want to send the query. By default, the interface from which to send the query is determined by the IP routing table.
-n (NT2003)
Displays IP addresses in numeric format.
-r retry_count (NT2003)
Specifies the neighbor query retry limit (default 3).
-t timeout_count (NT2003)
Specifies how long in seconds to wait for a neighbor query reply (default 4).
