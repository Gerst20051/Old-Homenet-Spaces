PBADMIN

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Administers phone books.

Syntax

PBADMIN /?

PBADMIN /N phonebook_name [/P data_filepath\data_filename] [/R region_filepath\region_filename]

PBADMIN /I phonebook_name [/P data_filepath\data_filename | /R region_filepath\region_filename]

PBADMIN /O phonebook_name server_name user_name password

PBADMIN /B phonebook_name

Starts Phone Book Administrator:

PBADMIN

Parameters
password (NTXP)
Specifies the password for the user account on the PBS server.
server_name (NTXP)
Specifies the name of the server to which the phone book is published.
user_name (NTXP)
Specifies a user account with FTP permissions on the Phone Book Service (PBS) server.
Switches
/? (NTXP)
Displays help.
/B phonebook_name (NTXP)
Publishes a phone book. The phonebook_name parameter specifies the name of the phone book to publish. Before you can use the /B switch, you must configure the phone book with the /O switch.
/I phonebook_name (NTXP)
Imports data into a phone book (.mdb) from a region (.txt) file or a phone book (.pbk or .txt) file. Specifies the phone book into which you want to import data.
/N phonebook_name (NTXP)
Creates a phone book (.mdb). Specifies the name of the file to create.
/O phonebook_name (NTXP)
Configures options for a phone book. The PhoneBookName specifies the name of the phone book for which you want to set options.
/P data_pathname/data_filename (NTXP)
Specifies the location and the name of the phone book file (.pbk or .txt) you want to import into a phone book.
/R region_pathname/region_filename (NTXP)
Specifies the location and the name of the region file you want to import into a phone book.
Related

none.

Notes

A phone book name cannot consist of more than eight characters, cannot consist of all digits, and must not contain a space or any of the symbols: ! , ; * = / \ : ? ' " < > | . & % {} [] @ ( ) ´ ` ~

If you specify a path, you must specify the full path. You cannot use spaces in the path to the phone book, to a phone book file, or to a region file.
