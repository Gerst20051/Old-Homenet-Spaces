PERFMON

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Allows you to open a Windows XP Performance console configured with settings files from Windows NT 4.0 version of Performance Monitor.

Syntax

PERFMON [file_name] [/HTML converted_file settings_file | /HTMLFILE: converted_files settings_file]

Parameters
file_name (NTXP)
Specifies the name of the settings file.
Switches
/HTML converted_files settings_file (NTXP)
/HTMLFILE: converted_files settings_file (NT2003)
Specifies the name of the converted files, and the name of the original Windows NT 4.0 settings file.
