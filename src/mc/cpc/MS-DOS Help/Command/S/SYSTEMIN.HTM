SYSTEMINFO

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays detailed configuration information about a computer and its operating system, including operating system configuration, security information, product ID, and hardware properties, such as RAM, disk space, and network cards.

Syntax

SYSTEMINFO /?

SYSTEMINFO [/S computer [/U [domain\]user [/P password]]] [/FO format} [/NH]

Parameters

none.

Switches
/? (NTXP)
Displays help.
/FO format (NTXP)
Specifies the format to use for the output. Valid values are:
  • TABLE
  • LIST The default
  • CSV
/NH (NTXP)
Suppresses column headers in the output. Valid when the /fo is set to TABLE or CSV.
/P password (NTXP)
Specifies the password of the user account that is specified in /u.
/S computer (NTXP)
Specifies the name or IP address of a remote computer (do not use backslashes). The default is the local computer.
/U [domain\]user (NTXP)
Runs the command with the account permissions of the user specified by User or Domain\User. The default is the permissions of the current logged on user on the computer issuing the command.
