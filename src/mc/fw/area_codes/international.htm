The North American Numbering Plan - NANP

Facts about the NANP:

The North American Numbering Plan serves 19 countries including the United States
All 19 countries part of the NANP have the same telephone numbering format: YXX-YXX-XXXX where Y can be any digit from 2 through 9 and X can be any digit
All countries part of the NANP have country code 1(one)
Dialing between 2 countries that are part of the North American Numbering plan (e.g. United States to Canada) is the same as dialing within a NANP country: 1 + 3 digit area code + 7 digit telephone number*

*Caution: the above can result in placing unaware international calls charged at high rates, so make sure you first identify an unknown area code before you dial the number.
