PORTUAS

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Merges a LAN Manager 2.x user accounts database into an existing Windows 2000 user accounts database.

Syntax

PORTUAS -f filename [-u username] [-v] [/codepage codepage] [/log filename]

Parameters

none.

Switches
/codepage codepage (NT3.5)
Specifies the OEM code page the LAN Manager 2.x NET.ACC file is in.
-f filename (NT3.1)
Specifies the LAN Manager 2.x NET.ACC file.
/log filename (NT3.5)
Specifies a file to log results.
-u username (NT3.1)
Specifies a single user or group to restore.
-v (NT3.1)
Displays all messages (verbose).
