%MSDOSDATA%

Description | Syntax | Related | Notes | Examples | Availability

Specifies the path to the MSAV (Anti-Virus for MS-DOS) MSAV.INI initialization file.

Also specifies the location of MSBACKUP (Backup for MS-DOS) configuration information, backup sets, and catalogs.

Syntax

%MSDOSDATA%

Related

MSAV

MSBACKUP

Notes

none.

Examples 
    MSDOSDATA=C:\BACKUP
