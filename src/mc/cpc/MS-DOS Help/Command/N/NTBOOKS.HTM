NTBOOKS

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Accesses online Windows NT manuals. It searches for Windows NT manuals at the last used location, but does not display the path to the location. If the location was a CD-ROM drive and the CD with the books is not in the drive, you are prompted to insert the CD.

Syntax

NTBOOKS [/s] [/w] [/n: path]

Parameters

none.

Switches
/n: path (NT3.51)
Specifies the path to the online books. Normally, the last used path is remembered and used.
/s (NT3.51)
Used from a Windows NT workstation to access Windows NT server documentation.
/w (NT3.51)
Used from a Windows NT server to access Windows NT workstation documentation.
