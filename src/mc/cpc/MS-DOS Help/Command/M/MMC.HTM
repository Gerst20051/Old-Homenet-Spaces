MMC

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Opens Microsoft Management Console (MMC).

Syntax

MMC /?

MMC Path\filename.msc [/a] [/32] [/64]

Parameters
Path\filename.msc (NTXP)
Starts MMC and opens a saved console. You need to specify the complete path and file name for the saved console file. If you do not specify a console file, opens a new console.
You can use environment variables to create command-lines or shortcuts that do not depend on the explicit location of console files.
Switches
/? (NTXP)
Displays help.
/32 (NTXP)
Opens the 32-bit version of MMC (MMC32). When running Windows XP 64-Bit Edition, you can run 32-bit snap-ins by opening MMC with this command-line option.
/64 (NTXP)
Opens the 64-bit version of MMC (MMC64). Use only if you are running Windows XP 64-Bit Edition, 32-Bit is the default.
/a (NTXP)
Opens a saved console in author mode. Used to make changes to saved consoles.
When consoles are opened with this option, they are opened in author mode, regardless of their default mode. This does not permanently change the default mode setting for files. When you omit this option, console files are opened according to their default mode settings.
