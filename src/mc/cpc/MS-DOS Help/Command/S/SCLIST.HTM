SCLIST

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Display NT Services.

Syntax

SCLIST [-r] [-s] [computer_name]

Parameters
computer_name (NT4)
The computer running the services, default %COMPUTERNAME%.
Switches
-r (NT4)
Display only running services.
-s (NT4)
Display only stopped services.
Related

NET HELP - Manage network resources.
SC - Service Control.
MODE CON - Configure width of CMD window.
