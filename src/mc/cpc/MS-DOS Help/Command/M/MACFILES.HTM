MACFILE SERVER

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Change the logon message and limit sessions.

Syntax

MACFILE SERVER /?

MACFILE SERVER [/loginmessage:message] [/maxsessions:nn | unlimited] [/server:\\Computer]

Parameters

none.

Switches
/? (NTXP)
Displays help.
/loginmessage:message (NTXP)
Changes the message Macintosh users see when logging on to the File Server for Macintosh server. The maximum number of characters for the logon message is 199. If omitted, the message for the server remains unchanged. To remove an existing logon message, leave the Message variable blank.
/maxsessions:nn | unlimited (NTXP)
Specifies the maximum number of users who can simultaneously use File and Print Servers for Macintosh. If omitted, the setting for the server remains unchanged.
/server:\\Computer (NTXP)
Specifies the server on which to change parameters. If omitted, the operation is performed on the local computer.
