DOS v3.10

Notes | Added | Changed | Removed | Files
  • Fixed some bugs.
  • OEM various manufacturers, released in November 1984.
  • Added generic network services/support (file sharing, locking, redirecting).
  • Added DOS functions 44H subfunctions 09H and 0aH for network support, DOS functions 5eH (network printer) and 5fH (network device redirection).
  • Limit programs that use FCBs to only open 4 FCBs at once. (Open a fifth and SHARE would close the oldest file control block).
  • AT&T DOS, cluster-size and file allocation table structures done differently.
  • "Pretender" commands introduced:
    1. JOIN
    2. SUBST
  • Commands enhanced:
    1. LABEL
    2. LINK
    3. TREE
