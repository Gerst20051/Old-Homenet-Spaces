DOS v3.30

Notes | Added | Changed | Removed | Files
  • Released with the IBM Personal System/2 (IBM PS/2) line and compatible with all PCs, August 1987 (announced April 1987).
  • TI laptop ROM, 1990 (version 3.30R).
  • Added support for 1.44Mb 3.5" High-density (HD) diskettes.
  • Added support for four (4) serial ports (was two), and at higher speeds.
  • Added complex system for extra national language support.
  • Added the ability to use logical partitions greater than 32 MB total. Each drive letter in the logical partition is limited to 32 MB. The primary DOS partition is limited to 32 MB.
  • Default number of disk buffers now based on system's configuration, from:
    1. PC and XT: two.
    2. AT: three.
    to:
    1. minimal RAM and no high-density floppies, 3.5 diskettes, or hard disks: two.
    2. 512K RAM: fifteen.
  • Added DOS functions:
    1. 44H subfunction 0cH prepare and select code page fonts.
    2. 66H select code page fonts (when NLSFUNC is resident).
    3. 67H increase the maximum number of file handles for a process.
    4. 68H force DOS to update a file to disk ("commit" a write).
  • Commands introduced:
    1. 4201.CPI
    2. 5202.CPI
    3. @
    4. APPEND
    5. CALL
    6. CHCP
    7. COUNTRY.SYS
    8. DISPLAY.SYS, national language and font support.
    9. "documented" environment variables
    10. FASTOPEN
    11. KEYB
    12. KEYBOARD.SYS
    13. LCD.CPI
    14. NLSFUNC
    15. PRINTER.SYS, national language and font support.
  • Commands enhanced:
    1. DATE and TIME set the battery-powered clock (CMOS Memory) on AT and PS/2.
    2. ATTRIB spans multiple files and directories.
    3. BACKUP is faster and will format diskettes on-the-fly.
    4. FDISK supports multiple "extended" (logical) DOS partitions on a hard disk.
    5. RESTORE allows selection by time and date of modification; stops "accidently" overwritting IBMBIO.COM, IBMDOS.COM, and COMMAND.COM.
    6. XCOPY supplies ERRORLEVEL indication for use in batch files.
    7. KEYB replaces KEYBxx and uses Keyboard.Sys.
    8. NLSFUNC installs code page font-switching.
    9. CHCP selects/displays current active code page.
    10. MODE Redirect Printing
    11. MODE supports code page selection and switching.
    12. ECHO
    13. GRAFTABL
