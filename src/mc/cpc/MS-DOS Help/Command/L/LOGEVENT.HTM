LOGEVENT

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Write text to the NT event log.

Syntax

LOGEVENT [-m \\MACHINENAME] [-s [S|I|W|E|F]] [-c nnnnn] "Event Text"

Parameters
"Event Text" (NT4)
Text message you want logged.
Switches
-c nnnnn (NT4)
Category - A Number between 0 and 65536 This can be used to Filter the event log view (default = "none").
-m \\MACHINENAME (NT4)
Name of the machine where log resides.
-s [S|I|W|E|F] (NT4)
Severity is one of:
  • S Success
  • I Information
  • W Warning
  • E Error
  • F Failure
