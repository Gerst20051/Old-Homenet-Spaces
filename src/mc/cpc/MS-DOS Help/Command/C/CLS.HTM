Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Clears the screen.

The cleared screen shows only the command prompt and cursor.

Syntax

CLS /?

CLS

Parameters

none

Switches
/? (NTXP)
Display help.
Related

none

Notes
  • Windows 95 Note
Windows 95 Note

Close an open Batch-window (DOS-Prompt) after running a batch program.

The window remains open when there is text to display.

Clear the screen (CLS) and the window will close.
