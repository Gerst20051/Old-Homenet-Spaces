UNLODCTR

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Removes Performance counter names and Explain text for a service or device driver from the system registry.

Syntax

UNLODCTR /?

UNLODCTR [\\computer_name] driver_name

Parameters
\\computer_name (NTXP ONLY)
If computer_name omitted, uses the local computer by default.
driver_name (NTXP)
Removes the Performance counter name settings and Explain text for driver or service driver_name from the Windows XP system registry.
Switches
/? (NTXP)
Displays help.
