UNLOCK

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Unlocks a drive, disabling direct disk access for an application.

Syntax

UNLOCK [drive:]

Parameters
drive: (Win95)
Specifies the drive to unlock.
Switches

none.

Related

Can be used to unlock a disk drive for direct disk access after using commands like: Undelete
To lock a disk drive for direct disk access use: LOCK
