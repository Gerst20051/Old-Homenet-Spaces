SIZER

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Used by the MEMMAKER memory-optimization program to determine the size in memory of device drivers and memory-resident programs.

During optimization, MEMMAKER adds SIZER.EXE to the beginning of each CONFIG.SYS or AUTOEXEC.BAT command that starts a driver or program. When the optimization process is complete, MEMMAKER removes SIZER.EXE from each command-line.

This program is used exclusively by MEMMAKER.
