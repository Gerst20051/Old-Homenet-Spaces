NET SESSION

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Lists or disconnects sessions between the computer and other computers on the network.

Syntax

NET SESSION [\\computer_name] [/DELETE] [/Y]

Displays information about all sessions with the computer of current focus:

NET SESSION

Parameters
\\computer_name (NT3.1)
Lists the session information for the named computer.
Switches
/DELETE (NT3.1)
Ends the session between the local computer and computername, and closes all open files on the computer for the session. If computername is omitted, all sessions are ended.
/Y (NT4 ONLY)
Execute command without prompts.
Related

none.

Notes

Displays incoming connections only, in other words it must be run on the machine that is acting as the server.
