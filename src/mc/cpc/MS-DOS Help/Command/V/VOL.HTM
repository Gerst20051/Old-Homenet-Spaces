VOL

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays the disk volume label and serial number, if the disk has them.

Syntax

VOL /?

Display the volume label of the disk in the current drive:

VOL

VOL [drive:]

Parameters
drive: (v2.0 Win95 NT3.1)
Specifies the drive that contains the disk for which you want to display the volume label and serial number (serial numbers for disks formatted starting with v4.0
Switches
/? (NTXP)
Displays help.
Related

For information about the /V[:label] switch, see the FORMAT command.
For information about assigning a volume label, see the LABEL command.
Equivalent Linux BASH commands:
hostname - Print or set system name.
uname - Print system information.
