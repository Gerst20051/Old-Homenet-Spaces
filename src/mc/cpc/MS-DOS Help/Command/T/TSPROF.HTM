TSPROF

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Copies the user configuration information, which is displayed in the Terminal Services extensions to Local Users and Groups and Active Directory Users and Computers, from one user to another. Tsprof can also set the profile path for a user.

Syntax

TSPROF /?

TSPROF /copy [/domain:domain_name | /local] [/profile:profile_path] source_user destination_user

TSPROF /q[uery] [/domain:domain_name | /local] user_name

TSPROF /update [/domain:domain_name | /local] /profile:profile_path user_name

Parameters
destination_user (NT2000)
Specifies the name of the user to whom you want to copy the user configuration information.
source_user (NT2000)
Specifies the name of the user from whom you want to copy the user configuration information.
user_name (NT2000)
Specifies the name of the user for whom you want to update or query the server profile_path.
Switches
/? (NT2000)
Displays help.
/copy (NT2000)
Copies user configuration information from source_user to destination_user and updates the profile_path information for destination_user to profile_path. Both source_user and destination_user must either be local or must be in domain domain_name.
/domain:domain_name (NT2000)
Specifies the name of the domain in which the operation is applied.
/local (NT2000)
Applies the operation only to local user accounts.
/profile:profile_path (NT2000)
Specifies the profile path as displayed in the Terminal Services extensions in Local Users and Groups and Active Directory Users and Computers.
/q[uery] (NT2000)
Displays the current profile_path of the user for whom you want to query the server profile_path.
/update (NT2000)
Updates profile_path information for user_name in domain domain_name to Path.

Related

none.

Notes

none.

Examples

To copy user configuration information from LocalUser1 to LocalUser2, type:

    TSPROF /copy /local LocalUser1 LocalUser2

To set the Terminal Services profile path for LocalUser1 to a directory called "c:\profiles," type:

    TSPROF /update /local /profile:c:\profiles LocalUser1

none.
