IPSECCMD QUERY MODE

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

You can use query mode to display data from the IPSec security policies database.

Syntax

IPSECCMD show /?

IPSECCMD [\\computer_name] show queries

Parameters
\\computer_name (NTXP)
Specifies, by name, the remote computer for which you want to display data. Defaults to local computer.
queries (NTXP)
  • All all the following
  • Auth main mode authentication methods
  • Filters main/quick mode filters
  • Policies main/quick mode policies
  • Sas main/quick mode security associations (SAs)
  • Stats Internet Key Exchange (IKE) and IPSec
show (NTXP)
Indicates that Ipseccmd should run in query mode.
Switches
/? (NTXP)
Displays help.
Related

IPSECCMD DYNAMIC/STATIC MODE

Notes

Cannot be used to configure rules on computers running Windows 2000.
