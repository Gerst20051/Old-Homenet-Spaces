NET LOGON

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Identifies you as a member of a workgroup.

Syntax

NET LOGON [user [password | ?]] [/DOMAIN:name] [/YES] [/SAVEPW:NO]

If you would rather be prompted to type your user name and password instead of specifying them in the command-line:

NET LOGON

Parameters
password (v7.00 Win95)
The unique string of characters that authorizes you to gain access to your password-list file. The password can contain up to 14 characters.
? (v7.00 Win95)
Specifies that you want to be prompted for your password.
user (v7.00 Win95)
Specifies the name that identifies you in your workgroup. The name you specify can contain up to 20 characters.
Switches
/DOMAIN:name (v7.00 Win95)
Specifies that you want to log on to a Microsoft Windows NT or LAN Manager domain.
Specifies the Windows NT or LAN Manager domain you want to log on to.
/SAVEPW:no (v7.00 Win95)
Carries out the NET LOGON command without prompting you to create a password-list file.
/YES (v7.00 Win95)
Carries out the command without first prompting you to provide information or confirm actions.
