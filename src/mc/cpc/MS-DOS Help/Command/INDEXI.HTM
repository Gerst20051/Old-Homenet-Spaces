  1. Error Check Syntax:
    IF [NOT] ERRORLEVEL nnn command
  2. IF [NOT] DEFINED variable command
  3. IF CMDEXTVERSION nnn command
  4. String syntax:
    IF [/I] [NOT] string1==string2 command
  5. IF [/I] item1 compare-op item2 command
  6. IF [/I] item1 compare-op item2 (command) ELSE (command)
  7. File syntax:
    IF [NOT] EXIST filename command
  8. IF [NOT] EXIST filename (command) ELSE (command)
