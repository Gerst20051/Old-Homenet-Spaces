MAPISEND

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Send email from the command-line.

Syntax

MAPISEND -u "profile" -p password -r recipient -s "subject" -m "text message" [-c] [-f] [-i] [-v]

MAPISEND -u "profile" -p password -r recipient -s "subject" -t text file [-c] [-f] [-i] [-v]

Parameters

none.

Switches
-c (NT4)
cc: list.
-f (NT4)
File Attachment - path and file name(s).
-i (NT4)
Interactive login (prompts for profile and password).
-m "text message" (NT4)
Text message.
-p password; (NT4)
Password.
-r recipient; (NT4)
"recipient" is one or more recipient(s). If more than one recipient - separate with ';'.
-s "subject" (NT4)
"subject" is the subject line.
-t text file (NT4)
Text file.
-u "profile" (NT4)
"profile" is the profile name (user mailbox) of sender.
-v (NT4)
Generates verbose output.
Related

MAPISEND requires MAPI - i.e the MS Exchange/Outlook client needs to be installed.
BLAT (3rd-party).
CommandMail (3rd-party).
Equivalent Linux BASH commands:
sendmail.

Notes

none.

Examples 
    MAPISEND -u "john doe" -p MyPassword -r billg@beast.com -s "Test message" -t c:\MyMail.txt
Errorlevels

none.
