  1. MINUMUM SYSTEM REQUIREMENTS
    1. http://support.microsoft.com/default.aspx?scid=kb;en-us;Q182751
    2. CPU: 486DX 66MHz
    3. RAM: 16MB
    4. HARD DRIVE SPACE: 140-255MB
  2. Microsoft Windows 98 (code-name "Memphis"), beta 1 begins January 6 1997.
  3. Microsoft Windows 98 beta 2 begins December 15 1997.
  4. OEM (Pre-Installed) with full FAT32 (support disks up to 2Tb), USB Support and Integrated Internet Explorer (IE) 4.0 (Install CD-ROM is now 'Bootable'), released in June 1998.
  5. Retail version for computers without DOS available.
  6. Previous Windows versions Upgrade available (Win98UG).
  7. Educational Pack: Combined Win98 and NT4.0 for Teachers/Students only.
  8. Plus Pack: 'Extras" that could have been included on 98 CD.
  9. Service Pack 1 (SP1): Major Bug Fixes plus IE 4.01 SP1, 1999.
  10. Windows 98SE OEM (Pre-Installed - Second Edition): Integrated Internet Explorer 5.0, released in May 5 1999.
  11. Windows 98SE Retail, for computers without MS-DOS or Windows, 1999.
  12. Previous Windows versions SE Upgrade available, 1999.
  13. Educational Pack: Combined Win98SE and NT4.0(SP-5) for Teachers/Students only, 1999.
  14. Successor (code-name "Neptune") cancelled in 1999, replaced by "Whistler".
