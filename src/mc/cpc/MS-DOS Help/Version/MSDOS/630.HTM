PC DOS v6.3

Notes | Added | Changed | Removed | Files
  • PC DOS 6.3 (released April 1994) contains many new and enhanced tools!
    • Central Point Backup with scheduler and tape support
    • Central Point Undelete with three levels of protection
    • Memory management improvements and easy-to-use optimizer (RAMSETUP) that sets up your computer's memory for you
    • IBM AntiVirus, Virus protection including detection and removal
    • Powerful and flexible full feature editor
      • allows you to load and work with multiple files
      • contains global search and replace capabilities
      • includes math, drawing and sorting features
    • Multiple configurations so each time you start your computer, you can choose the configuration you want to use
    • Increased storage capacity compression (SSTOR) for both diskettes and hard disks
      • Easy-to-use DOS menu interface for compression
      • Increased storage capacity for both diskettes and hard disks
      • Safety and disk optimizing utilities
      • Password protection
      • Capability to transport your compressed data to other DOS systems
      • Compression preview to display expected compression ratio
      • Utility to uncompress your data
