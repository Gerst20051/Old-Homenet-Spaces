WINMSD

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Windows NT diagnostics

Reports: Memory use, Services, Devices, IRQ's Ports, Environment variables, Network (rights, transport, stats), Hardware including Display adapter.

Syntax

WINMSD \\computer_name

WINMSD [[/a] | [/s]] [[/f] | [/p]]

Open the GUI with details of the computer you are logged into:

WINMSD

Parameters
\\computer_name (NT4)
Shows all permissions and group memberships.
Switches
/a (NT4)
All details.
/f (NT4)
Send output to a file "computername.txt" in the current directory. If you have the resource kit WINMSDP allows more control over Spooling output to file.
/p (NT4)
Send output to a printer.
/s (NT4)
Summary details only.
Related

MSINFO32 - Windows NT diagnostics.
WINMSDP - Windows NT Diagnostics II.

Notes

Hot keys within the GUI:

SHIFT-F2: copies the current tab to the clipboard.

F2: copies a summary of the current tab to the clipboard.
