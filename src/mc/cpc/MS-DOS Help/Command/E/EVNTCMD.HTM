EVNTCMD

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays SNMP events.

Syntax

EVNTCMD [/? | /h]

EVNTCMD [/n] [/s sysname] [/v number] file_name]

Parameters
file_name (NT2003)
Specifies, by name, the configuration file that contains information about the translation of events to traps and trap destinations you want to configure.
Switches
/h (NT2000)
/? (NT2000)
Displays help.
/n (NT2000)
Specifies not to restart the SNMP service when it receives trap changes.
/s sysname (NT2000)
Specifies the target system name for configuration.
/v number (NT2000)
Specifes the level of verbosity for evntcmd. 0=silent, 10=detailed.
