WINDOWS FOR WORKGROUPS 3.1

Notes | Added | Changed | Removed | Files
  1. Released October 27 1992.
  2. DOS Graphical User Interface (GUI).
  3. Integrates networking and workgroup functionality directly into Windows 3.1.
  4. Allows common activities of sending electronic mail, scheduling group meetings, sharing files and printers, managing calendars, and working together on groups projects.
  5. It can provide networking capabilities on its own, or it can be used as a client on an existing Local Area Network (LAN).
  6. DOS upgrade version available (WFW31AD).
  7. Windows 3.1 upgrade version available (WFW31AW).
