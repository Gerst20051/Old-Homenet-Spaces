What are those image filters?

  • Faded - Sets the amount of Opacity (transparency).
  • Circular - make the images round and faded looking.
  • Blur - Makes object appear to be moving.
  • Drop Shadow - Create a drop shadow.
  • Flip Horizontal - Flips the object Horizontally (mirror).
  • Flip Vertical - Flips the object vertically.
  • Glowing - Adds a radiant glow around the object.
  • Gray Scale - Turns object to grayscale.
  • Invert - Reverses the Saturation, Hue, and Brightness of an object.
  • Lighting - Project a light source onto the object.
  • Masking - Create a transparent mask from the object.
  • Shadow - Create a shadow with feathered edges.
  • Wavy - Create a sine wave distortion along the vertical axis.
  • BW Negative - Makes object into a B/W negative.
