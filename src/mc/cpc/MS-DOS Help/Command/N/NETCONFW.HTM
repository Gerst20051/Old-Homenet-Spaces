NET CONFIG WORKSTATION

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays or changes settings for the Workstation service.

Syntax

NET CONFIG /?

NET CONFIG WORKSTATION [/CHARCOUNT:bytes] [/CHARTIME:msec] [/CHARWAIT:sec]

Display the current configuration for the Workstation service:

NET CONFIG WORKSTATION

Parameters

none.

Switches
/? (NTXP)
Displays help.
/CHARCOUNT:bytes (NT3.1 through NT4)
Specifies the amount of data Windows collects before sending the data to a communication device. If /CHARTIME:msec is also set, Windows acts on whichever option is satisfied first. The range is 0-65535 bytes; the default is 16 bytes.
/CHARTIME:msec (NT3.1 through NT4)
Sets the number of milliseconds Windows collects data before sending the data to a communication device. If /CHARCOUNT:bytes is also set, Windows acts on whichever option is satisfied first. The range is 0-655350000 milliseconds; the default is 250 milliseconds.
/CHARWAIT:sec (NT3.1 through NT4)
Sets the number of seconds Windows waits for a communication device to become available. The range is 0-65535 seconds; the default is 3600 seconds.
Related

none.

Notes

Available only on Windows NT Servers.
