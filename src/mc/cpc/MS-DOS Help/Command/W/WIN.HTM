WIN

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Starts the Windows Graphical User Interface (GUI).

Syntax

WIN [/D[[:]F][[:]M][[:]N][[:]S][[:]V][[:]X]]

WIN [[/MQ] | [/W] | [/WX] | [/Z] | [ : ]]

Parameters

none

Switches
: (Win30)
UNDOCUMENTED Disables display of startup logo.
/D (v7.00 Win95)
Used for troubleshooting when Windows does not start correctly.
:F (v7.00 Win95)
Turns off 32-bit disk access. Equivalent to SYSTEM.INI file setting: 32BitDiskAccess=FALSE.
:M (v7.00 Win95)
Enables Safe mode. This is automatically enabled during Safe start (function key F5).
:N (v7.00 Win95)
Enables Safe mode with networking. This is automatically enabled during Safe start (function key F6).
:S (v7.00 Win95)
Specifies that Windows should not use ROM address space between F000:0000 and 1 MB for a break point. Equivalent to SYSTEM.INI file setting: SystemROMBreakPoint=FALSE.
:V (v7.00 Win95)
Specifies that the ROM routine will handle interrupts from the hard disk controller. Equivalent to SYSTEM.INI file setting: VirtualHDIRQ=FALSE.
:X (v7.00 Win95)
Excludes all of the adapter area from the range of memory that Windows scans to find unused space. Equivalent to SYSTEM.INI file setting: EMMExclude=A000-FFFF.
/MQ (v7.00 Win95)
UNDOCUMENTED Fast soft-reboot without prompting.
/W (v7.00 Win95)
UNDOCUMENTED Warm-reboot with prompt.
/WX (v7.00 Win95)
UNDOCUMENTED Warm-reboot without prompt.
/Z (v7.00 Win95)
UNDOCUMENTED Forces ATX power supply to soft-power-off without prompting.
Related

Replaces WIN386 used in previous Windows versions
