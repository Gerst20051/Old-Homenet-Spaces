MSINFO32

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Windows NT diagnostics; Reports: Memory use, Drivers, DLL versions, Audio,Video and Print settings.

Syntax

MSINFO32 /?

MSINFO32 [/c] [/categories category_id] [/category category_id] [/computer computer_name] [/msinfo_file] [/nfo filename.nfo | /s filename.nfo] [/pch] [/report filename.txt] [/showcategories]

Parameters

none.

Switches
/? (NTXP)
Displays help.
/c (NT2000 ONLY)
List the version, date, and build of each DLL in a user-specified folder and determine whether it's loaded in memory.
/msinfo_file (NT2000 ONLY)
Open the specified .nfo or .cab file.
/nfo filename.nfo ( /s filename.nfo NT2000)
Output an .nfo file with the specified filename.
/report filename.txt (NT2000)
Output a text file with the specified filename.
/computer computer_name (NT2000)
Get details from a remote computer_name.
/categories category_id (NT2000)
Starts System Information with only the specified category or categories displayed. It also limits the output to the selected category or categories. Use /showcategories to display a list of available category IDs.
/category category_id (NT2000)
Starts System Information with the specified category selected. Use /showcategories to display a list of available category IDs.
/pch (NTXP)
Displays the history view.
/showcategories (NTXP)
Starts System Information with all available category IDs displayed.
Related

WINMSDP - Windows NT Diagnostics II.

Notes

The GUI interface will open if no options are specified.

Early versions of MSInfo do not support all the switches listed.

MSInfo32 is typically started from Help,About rather than the command-line, if not in the system path, MSInfo32 can usually be found in: C:\Program Files\Common Files\Microsoft Shared\MSInfo.

Generation of the text file can take some time, depending on the complexity of the system.

If you have problems getting MSInfo32 to run, check permissions on the key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Shared Tools\MSInfo\.

Examples 
    msinfo32 /c [My DLL's] c:\mydir

none.
