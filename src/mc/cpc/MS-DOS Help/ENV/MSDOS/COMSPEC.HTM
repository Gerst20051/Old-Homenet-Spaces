%COMSPEC%

Description | Syntax | Related | Notes | Examples | Availability

Specifies the location of the command prompt interpreter (COMMAND.COM or CMD.EXE) on the floppy/hard disk when the transient portion needs to be reloaded into memory again.

Syntax

%COMSPEC%

Related

COMMAND.COM

CMD.EXE

Notes

none.

Examples 
    COMSPEC=C:\WINDOWS\SYSTEM32\CMD.EXE
Availability
Static
