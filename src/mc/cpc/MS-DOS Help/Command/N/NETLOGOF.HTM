NET LOGOFF

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Breaks the connections between your computer and the shared resources to which it is connected.

Syntax

Parameters

none.

Switches
/YES (v7.00 Win95)
