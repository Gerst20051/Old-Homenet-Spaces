Acknowledgements

I have always wanted all of this information in ONE place. Which options? versions? Was that Windows NT4? 95? 2K? DOS3.3? However, I was forever jumping from site-to-site or file-to-file.

I wanted:

  1. Simple HTML (no tables) <DL>/<DT>/<DD> and <PRE>
  2. A single file per command, containing ALL its versions and which options where added with which version.
  3. Hyperlink all command parameters and options.
  4. Both syntax-only and descriptive lists, so I have both.
  5. Additional examples (thanks Benny).
  6. Equivalent BASH commands (thanks Simon).
  7. Small (less than 50Kb) HTML pages (ONLY 9 of the files exceed this!)
  8. And Later: Searchable, so I built a Compiled HtmlHelp (.CHM) version.

Microsoft has tried to made it easy to know some of the commands... I want to thank Microsoft, IBM and the people (some of which are referenced below) that have tried to keep up with the MANY flavors of the commands and operating systems.
