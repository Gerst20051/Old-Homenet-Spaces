NET COMPUTER

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Adds or deletes computers from a domain database.

Syntax

NET HELP COMPUTER

NET COMPUTER \\computer_name {/ADD | /DEL}

Parameters
\\computer_name (NT3.1)
Specifies the computer to add or delete from the domain.
Switches
/ADD (NT3.1)
Adds the specified computer to the domain.
/DEL (NT3.1)
Removes the specified computer from the domain.
Related

none.

Notes

Available only on Windows NT Servers.
