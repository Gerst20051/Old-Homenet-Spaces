%PATH%

Description | Syntax | Related | Notes | Examples | Availability

Specifies a sequence of folders used to search for the location (in order from left-to-right) of a program (extensions .COM, .EXE, or .BAT), when it is not in the current folder.

Syntax

%PATH%

Related

PATH

Notes

none.

Examples 
    PATH=C:\WINDOWS;C:\DOS
