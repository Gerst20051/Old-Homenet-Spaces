DRIVERQUERY

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays a list of all installed device drivers and their properties.

Syntax

DRIVERQUERY /?

DRIVERQUERY [/s computer] [/u domain\user /p password] [/fo format] [/nh] [/v] [/si]

Parameters

none.

Switches
/? (NTXP)
Display help.
/fo format (NTXP)
Specifies the format to display the driver information. Valid values are: TABLE (default), LIST, and CSV.
/nh (NTXP)
Omits the header row from the displayed driver information. Valid when /fo is set to TABLE or CSV.
/p password (NTXP)
Specifies the password of the user account that is specified in /u.
/s computer (NTXP)
Specifies the name or IP address of a remote computer (do not use backslashes). The default is the local computer.
/si (NTXP)
Displays digital signature information for both signed and unsigned device drivers.
/u domain\user (NTXP)
Runs the command with the account permissions of the user specified by User or Domain\User. The default is the permissions of the current logged on user on the computer issuing the command.
/v (NTXP)
Specifies that detailed driver information be displayed.
