LODCTR

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Registers new Performance counter names and Explain text for a service or device driver, and saves and restores counter settings and Explain text.

Syntax

LODCTR /?

LODCTR [\\computer_name] filename [/s:filename] [/r:filename] [/t:service_name]

Parameters
\\computer_name (NTXP)
Registers the Performance counter name settings and Explain text provided in initialization file FileName. If you do not specify the ComputerName, uses the local computer by default.
Switches
/? (NTXP)
Displays help.
/r:filename (NTXP)
Restores Performance counter registry settings and Explain text from file FileName.
/s:filename (NTXP)
Saves Performance counter registry settings and Explain text to file FileName.
Caution: You will overwrite all Performance counter registry settings and Explain text, replacing them with the configuration defined in the file specified.
/t:service_name (NT2003)
Indicates that service service_name is trusted.
