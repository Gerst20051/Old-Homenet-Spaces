LPR

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

This connectivity utility is used to print a file to a computer running an LPD server.

Syntax

LPR /?

LPR [-S Server] -P Printer [-C Class] [-J Jobname] [-O Option] [-d] [-x] filename

Parameters
filename (NT3.5)
The name of the file to be printed.
Switches
/? (NTXP)
Displays help.
-C Class (NT3.5)
Specifies the content of the banner page for the class.
-d (NTXP)
Specifies that the data file must be sent before the control file. Use this switch if your printer requires the data file to be sent first. For more information, see your printer documentation.
-J Jobname (NT3.5)
Specifies the name of this job.
-O Option (NT2000)
Indicates the type of file:
  • -O l binary file (for example, PostScript)
  • The default is a text file
-P Printer (NT3.5)
Specifies the name of the printer for the desired queue.
-S Server (NT3.5)
Specifies the name or IP address of the computer that has the printer attached to it.
-x (NTXP)
Specifies that must be compatible with the Sun Microsystems operating system referred to as SunOS for releases up to and including 4.1.4_u1.
