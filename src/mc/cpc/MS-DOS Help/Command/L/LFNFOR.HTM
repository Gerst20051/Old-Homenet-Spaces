LFNFOR

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Enables/Disables Long file names when processing FOR commands.

Syntax

LFNFOR [ON | OFF]

To display the current setting:

LFNFOR

Parameters
ON | ( OFF Win95)
Enable (ON) / Disable (OFF) long filenames for use in the FOR command.

none.
