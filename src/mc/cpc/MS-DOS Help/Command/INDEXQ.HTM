Command line - Q

  1. QBASIC [/B] [/EDCOM] [/QBHELP] [/EDITOR] [/G] [/H] [/MBF] [/NOHI] [[/RUN] [drive:][path]filename]
  2. QUERY PROCESS /?
  3. QUERY PROCESS [{* | process_id | user_name | session_name | /id:nn | program_name}] [/server:server_name] [/system]
  4. QUERY SESSION /?

  6. QUERY TERMSERVER /?
  7. QUERY TERMSERVER [server_name] [/domain:domain] [/address] [/continue]
  8. QUERY USER /?
  9. QUERY USER [{user_name | session_name | session_id}] [/server:server_name]
