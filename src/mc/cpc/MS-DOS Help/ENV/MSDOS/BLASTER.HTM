%BLASTER%

Description | Syntax | Related | Notes | Examples | Availability

Specifies the SOUND BLASTER sound card settings.

Syntax

%BLASTER%

Related

none.

Notes
A is for address
210, 220, 230, 240, 250, 260, 280
I is for interrupt (IRQ)
2, 3, 5, 7, 10, 11, 12, 15
D is for DMA (8-bit)
0, 1, 3
? is for DMA (16-bit)
5, 6, 7
Examples 
    BLASTER=A220 I5 D1
