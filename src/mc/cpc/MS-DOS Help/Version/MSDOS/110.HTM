PC DOS v1.10 / MS DOS 1.25 (Z-DOS)

Notes | Added | Changed | Removed | Files
  • Released in May 1982.
  • OEM IBM Corporation PC-DOS (first release version).
  • OEM Zenith Computer Corporation Z-DOS.
  • Was standard for over a year.
  • Added support for 2-sided (double-sided), 8-sector, 320K (5.25") diskette drives (the end of the "flippy" where you could notch both edges of a diskette and flip it over to use the other side to store data).
  • Fundamental DOS services stabilized (debugged).
  • Uses DOS Functions 2eH and below.
  • Commands introduced:
    • DEL
    • EXE2BIN
    • REN
  • Commands enhanced:
    • CHKDSK
    • DATE, now internal COMMAND.COM command.
    • DEBUG
    • DISKCOMP
    • DISKCOPY
    • ERASE
    • FORMAT
    • LINK
    • MODE Configure Printer
    • MODE Configure Serial Port
    • MODE Redirect Printing
    • MODE Set Display Mode
    • TIME, now internal COMMAND.COM command.
