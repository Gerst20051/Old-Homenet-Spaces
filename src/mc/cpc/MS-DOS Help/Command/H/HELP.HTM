HELP (DOSHELP)

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Starts operating system command Help.

Syntax

HELP [/B] [/G] [/H] [/NOHI] [topic]

Parameters
topic (v5.0 Win95 NT3.1)
Specifies the command whose help topic you want to display.
Switches
/B (v5.0 Win95)
Allows use of a monochrome monitor with a color graphics card.
/G (v5.0 Win95)
Provides the fastest update of a CGA screen.
/H (v5.0 Win95)
Displays the maximum number of lines possible for your hardware.
/NOHI (v5.0 Win95)
Allows the use of a monitor without high-intensity support.
Related

For information about other online Help information, see the FASTHELP command.
Renamed in v5.0 from DOSHELP

Notes

none.

Examples

To display information about the COPY command, type:

    HELP COPY

To display the operating system Help table of contents, type HELP. From the table of contents, you can choose the command topic you want to view.

Errorlevels

none.
