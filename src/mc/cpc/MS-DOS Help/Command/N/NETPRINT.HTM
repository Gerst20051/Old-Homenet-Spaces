NET PRINT

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays print jobs and shared queues.

For each queue, the display lists jobs, showing the size and status of each job, and the status of the queue.

Syntax

NET HELP PRINT

NET PRINT \\computer_name[\printer] | port [/YES]

NET PRINT \\computer_name | port [job# [/DELETE | [/HOLD | /PAUSE] | [/RELEASE | /RESUME]] [/YES]

Parameters
\\computer_name (v7.00 Win95 NT3.1)
Is the name of the computer sharing the printer queue(s).
job# (v7.00 Win95 NT3.1)
Is the identification number assigned to a print job. A computer with one or more printer queues assigns each print job a unique number.
port (v7.00 Win95)
Specifies the name of the parallel (LPT) port on your computer that is connected to the printer you want information about.
printer (v7.00 Win95)
Specifies the name of the printer you want information about.
Switches
/DELETE (v7.00 Win95 NT3.1)
Cancels and removes a job from a queue.
/HOLD (NT3.1)
/PAUSE (v7.00 Win95)
Prevents a job in a queue from printing. The job stays in the printer queue, and other jobs bypass it until it is released.
/RELEASE (NT3.1)
/RESUME (v7.00 Win95)
Reactivates a job that is held.
/YES (v7.00 Win95)
Carries out the command without first prompting you to provide information or confirm actions.
