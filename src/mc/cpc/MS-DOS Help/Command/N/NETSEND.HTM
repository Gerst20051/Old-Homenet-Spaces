NET SEND

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Sends messages to other users, computers, or messaging names on the network.

Syntax

NET HELP SEND

NET SEND {name | * | /DOMAIN[:name] | /USERS} message

Parameters
message (NT3.1)
Text to be sent as a message.
name (NT3.1)
The username, computername, or messaging name to send the message to. If the name is a computername that contains blank characters, enclose the alias in quotation marks (" ").
* (NT3.1)
Sends the message to all the names in your group.
Switches
/DOMAIN:name (NT3.1)
Sends the message to all the names in the workstation domain. If name is specified, the message is sent to all the names in the specified domain or workgroup.
/USERS (NT3.1)
Sends the message to all users connected to the server.
Related

none.

Notes

The Messenger service must be running to receive messages.

You can send a message only to an name that is active on the network. If the message is sent to a username, that user must be logged on and running the Messenger service to receive the message.
