DYNALOAD

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Dynamically load device drivers from the command line.

Syntax

DYNALOAD filename [parameters]

Parameters
[drive:][\path\] (v7.0R1)
Specifies the location of the device driver program file.
filename (v7.0R1)
Specifies the name of the device driver program file.
parameters (v7.0R1)
Specifies any command-line information required by the device driver program.
