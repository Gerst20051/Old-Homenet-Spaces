FORCEDOS

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Starts the specified program in the MS-DOS subsystem. This command is necessary only for those MS-DOS programs not recognized as such by Windows.

Syntax

FORCEDOS [/d directory] filename [parameters]

Parameters
filename (NT3.1)
Specifies the program to start. If not in the current directory or Windows path, you must specify the drive letter or directories to the program.
parameters (NT3.1)
Specifies parameters to pass to the program.
Switches
/d directory (NT3.1)
Specifies the current directory for the specified program to use.
