IRFTP

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Sends files over an infrared link (requires an infrared device on the computer).

Syntax

IRFTP /?

IRFTP [/h] [[drive:] [path] filename [...]]

IRFTP /s

Parameters
drive:
path
filename [...] (NT2000)
The locations and names of the files you want to transmit over an infrared connection. If you do not specify a file, then the Wireless Link open files dialog box opens enabling you to select the files you want to send over the link.
Switches
/? (NTXP)
Displays help.
/h (NT2000)
Specifies hidden mode, which transmits the source files over the infrared link without displaying the Wireless Link dialog box.
/s (NT2000)
Opens the properties for Wireless Link.
