IISAPP

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Reports the process identifiers (PIDs) of currently running w3pwp.exe processes serving a particular application pool.

Syntax

IISAPP[.VBS] [/a application_pool_name | /p application_pool_id]

List all running applications:

IISAPP[.VBS]

Parameters

none.

Switches
/a application_pool_name (NT2003)
Specifies the name of a particular application pool.
/p application_pool_id (NT2003)
Specifies an application pool by its ID number.
