CHKNTFS

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Check the NTFS file system with CHKDSK.

Syntax

CHKNTFS /?

CHKNTFS /C drive: [...]

CHKNTFS /D

CHKNTFS /X drive: [...]

CHKNTFS /T[:time]

Display the status of the dirty bit for each drive:

CHKNTFS drive: [...]

Parameters
drive: (NT4)
Specifies the drive that contains the disk that you want to check.
Switches
/? (NTXP)
Display help.
/C (NT4)
Check - schedules to check at next reboot; chkdsk will run if the drive is dirty.
/D (NT4)
Restore the machine to the default behavior; all drives are checked at boot time and chkdsk is run on those that are dirty. This undoes the effect of the /X option.
/T[:time] (NT2000)
Changes the AUTOCHK initiation count down time to the specified amount of time in seconds. If time is not specified, displays the current setting.
/X (NT4)
Exclude a drive from the default boot-time check. Excluded drives are not accumulated between command invocations.
