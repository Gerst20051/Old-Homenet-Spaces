DOS v6.00

Notes | Added | Changed | Removed | Files
  • Released in March 1993 as a "catch-up" for Novell's DR-DOS 6.
  • Included many small changes to better support Windows 3.1.
  • Introduced menu-based (multi-boot) CONFIG.SYS configurations for system startup.
  • Allow debugging "single-step" through CONFIG.SYS allowing individual install or ignore for each entry.
  • DOS 3.3 (and upward) upgrade version (DOS6UG).
  • Supplimental (self-extracting) ZIP of earlier DOS versions removed files (DOS6SUPP.EXE) available.
  • DOSSHELL released separately (Microsoft thought there were too many disks).
  • Introduced DoubleSpace disk compression (from Stac Electronics).
  • Commands Introduced:
    1. CHOICE
    2. DBLSPACE, disk compression.
    3. DECOMP
    4. DEFRAG, rearrange layout of hard disk files (can improve the system efficiency).
    5. DELTREE
    6. FASTHELP, quick list of commands and their use.
    7. INTERLNK, link one computer to another via a cable, to transfer files or run programs on the connected computer.
    8. INTERSVR
    9. MEMMAKER, enhanced memory-management features to make even more memory available.
    10. MOVE
    11. MSAV, anti-virus.
    12. MSBACKUP, streamlined efficient hard disk backup (replaced slow and clumsy BACKUP).
    13. MSD
    14. POWER
    15. SIZER, used only by MEMMAKER.
    16. VSAFE, detect and remove computer viruses.
  • Commands improved:
    1. SMARTDRV.SYS, now caches floppy disks too.
  • Commands removed:
    1. ASSIGN command moved to supplemental programs.
    2. JOIN
    3. RECOVER
