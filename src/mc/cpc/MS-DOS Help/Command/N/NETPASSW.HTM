NET PASSWORD

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Changes your logon password.

Syntax

To change the password for your password-list file:

NET PASSWORD [oldpassword [newpassword]]

To change your password on a Windows NT or LAN Manager server or domain:

NET PASSWORD \\computer_name | /DOMAIN:name [user [oldpassword [newpassword]]]

Parameters
\\computer_name (v7.00 Win95)
Specifies the Windows NT or LAN Manager server on which you want to change your password.
newpassword (v7.00 Win95)
Specifies your new password. It can have as many as 14 characters.
oldpassword (v7.00 Win95)
Specifies your current password.
user (v7.00 Win95)
Specifies your Windows NT or LAN Manager user name.
Switches
/DOMAIN:name (v7.00 Win95)
Specifies that you want to change your password on a Microsoft Windows NT or LAN Manager domain.
Specifies the Windows NT or LAN Manager domain on which you want to change your password.
