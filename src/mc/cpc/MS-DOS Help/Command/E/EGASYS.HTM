EGA.SYS

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Saves and restores the display when the operating system Shell Task Swapper is used with EGA monitors. If you have an EGA monitor, you must install the EGA.SYS device driver before using Task Swapper. This device driver must be loaded by a DEVICE or DEVICEHIGH command in your CONFIG.SYS file.

Syntax

DEVICE=[drive:][\path\]EGA.SYS

Parameters
[drive:][\path\] (v5.0)
Specifies the location of the EGA.SYS file.
Switches

none.

Related

none.

Notes

If you are using a mouse on a system that has an EGA monitor, you can save memory by installing EGA.SYS before you install your mouse driver.
