LIBPATH

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Specifies the directories the OS/2 subsystem is to search for dynamic-link libraries. You can use this command only in your CONFIG.SYS file.

Syntax

LIBPATH=[drive:]path[;...]

Parameters
drive: path (NT3.1)
Specifies a drive, directory, and any subdirectories to search.
Switches

none.

Related

none.

Notes

none.

Examples

To instruct the OS/2 subsystem to look for dynamic-link libraries in the current directory and in the directory DYNLIB on your hard disk (drive C), add the following line to your C:\CONFIG.SYS file using an OS/2 editor:

    LIBPATH=.;c:\dynlib
Errorlevels

none.
