DOS v1.00

Notes | Added | Changed | Removed | Files
  • Came out for use with the 4.77MHz single-sided 5.25" floppy IBM 5150 Personal Computer (IBM PC) with 40Kb ROM, and 64Kb to 256Kb memory, 1981.
  • Used 86-DOS, (QDOS - Quick and Dirty Operating System) as base.
  • Well over 300 bugs found by IBM quality assurance, rewrite resulted in copyright by both IBM and Microsoft.
  • CP/M-like but for the 8088.
  • Supported 1-sided (single-sided), 8-sector, 160K (5.25") diskette drives only.
  • Fundamental DOS services were roughed out.
  • Introduced batch files.
  • All file access done through file control blocks.
  • Two executable types: COM and EXE. COM similar to CP/M: restricted to 64Kb of memory for code, data, and stack space.
  • Commands introduced:
    • BASIC
    • BASICA
    • CHKDSK
    • COMMAND
    • COMP
    • COPY
    • DATE
    • DEBUG
    • DIR
    • DISKCOMP
    • DISKCOPY
    • EDLIN
    • ERASE
    • FORMAT
    • LINK
    • MODE Configure Printer
    • MODE Configure Serial Port
    • MODE Redirect Printing
    • MODE Set Display Mode
    • PAUSE
    • REM
    • RENAME
    • SYS
    • TIME
    • TYPE
