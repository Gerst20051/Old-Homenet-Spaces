DCGPOFIX

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Restores the default Group Policy objects to their original state (that is, the default state after initial installation).

Caution: This tool can restore default domain policy and default domain controllers policy to their original state after installation. When you run DCGPOFIX, you will lose any changes made to these Group Policy objects.

Syntax

DCGPOFIX [/ignoreschema] [/target:{both | dc | domain}]

Parameters

none.

Switches
/ignoreschema (NT2003)
Ignores the Active Directory schema version number.
By specifying the /ignoreschema option, you can enable DCGPOFIX to work with different versions of Active Directory. However, default policy objects might not be restored to their original state. To ensure compatibility, use the version of DCGPOFIX that is installed with the current operating system.
/target:{both | dc | domain} (NT2003)
Specifies the target domain, domain controller, or both (default).
