MAKECAB

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Cabinet (.CAB) archive file maker.

Syntax

MAKECAB /?

MAKECAB [/V[n]] [/D var=value ...] [/L [dir]] source [destination]

MAKECAB [/V[n]] [/D var=value ...] /F directive_file [...]]

Parameters
destination (NTXP)
File name to give compressed file. If omitted, the last character of the source file name is replaced with an underscore (_) and used as the destination.
source (NTXP)
File to compress.
Switches
/? (NTXP)
Displays help.
/D var=value (NTXP)
Defines variable with specified value.
/F directive_file (NTXP)
A file (.DDF) with MakeCAB directives, there may be multiple files.
/L [dir] (NTXP)
Location to place destination (default is current directory).
/V[n] (NTXP)
Verbosity level (1..3).
