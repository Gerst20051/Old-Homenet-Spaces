DOS v3.20

Notes | Added | Changed | Removed | Files
  • Released with the laptop IBM Convertible PC, 1985.
  • First retail (others were all OEM) release of MS-DOS, January 1986.
  • Added support for 720K Low-density (LD) drives (3.5" hard-shell diskettes).
  • Made it more difficult to accidentally format a hard disk.
  • Expanded device driver IOCTL service to include DOS-blessed disk formatting, etc.
  • DOS function 44H extended to support Generic IOCTL and to support logical drives with subfunctions 0dH, 0eH, and 0fH.
  • Commands introduced:
    • DRIVER.SYS, phantom floppy driver.
    • REPLACE
    • XCOPY
  • Commands enhanced:
    • ATTRIB
    • COMMAND, environment size.
    • DISKCOMP
    • DISKCOPY
    • FORMAT
    • SELECT
    • GRAPHICS
    • SHELL
