VER

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays the operating system version number.

Syntax

VER /?

VER [/R]

Parameters

none.

Switches
/? (NTXP)
Displays help.
/R (v2.0 Win95)
UNDOCUMENTED Lists revision letter. And beginning with v5.0 and Win95 shows if COMMAND.COM is in the High Memory Area (HMA).
Related

none.

Notes
  • Windows' bundled DOS
  • Windows' version numbers

Windows' bundled DOS

No version of the DOS operating system after v6.22 has been released as a 'standalone version' which is why they report the Windows Version.

Main difference between v7.00 and v7.10 is Large Disk Support.

  • Windows 95 and 95A runs on v7.00
  • Windows 95B or C (aka OSR 2) runs on v7.10
  • Windows 98 and 98SE runs on v7.10
  • Windows ME runs on v8.00
  • Windows XP (aka Whistler) runs on ?
  • Windows 2000 (aka NT5) does not run on DOS.
Examples

When you enter the VER command, the operating system displays the message:

    MS-DOS Version 6.22

or

    Windows 98 [Version 4.10.1998]
