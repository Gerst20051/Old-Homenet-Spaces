PC DOS v2.10 / MS DOS v2.11

Notes | Added | Changed | Removed | Files
  • Released with the PCjr, March 1984.
  • OEM IBM Corporation PC-DOS.
  • OEM AT&T, Bull, Sony Corporations MS-DOS.
  • More colors onscreen in graphics mode.
  • Three-voice sound that could play chords.
  • Fixed some existing bugs in DOS 2.00; no new commands.
  • Added support for plug-in cartridges and the PCjr's half-height (5.25") diskette drives.
  • No country support
  • Enhancements to better allow conversion into different languages and date formats.
  • Bootable ROM for Tandy, 1988
  • DEC MSDOS supports quad-density, single-sided floppy.
