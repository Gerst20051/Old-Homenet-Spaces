WINMSDP

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Windows NT diagnostics II

Reports: Memory use, Services, Devices, IRQ's Ports, Environment variables, Network (rights, transport, stats), Hardware including Display adapter.

Syntax

WINMSDP /a | /d | /e | /i | /m | /n | /o | /p | /r | /s | /u | /w | /y

Parameters

none.

Switches
/a (NT4)
ALL prints everything.
/d (NT4)
Drives.
/e (NT4)
Environment.
/i (NT4)
Interrupt resources.
/m (NT4)
Memory.
/n (NT4)
Network.
/o (NT4)
OS version.
/p (NT4)
Port resources.
/r (NT4)
Drivers.
/s (NT4)
Services.
/u (NT4)
DMA resources.
/w (NT4)
Hardware.
/y (NT4)
Memory resources.
Related

WINMSD - Windows NT Diagnostics.

Notes

The output is very similar to WINMSD if a little more detailed.

The output will appear in a text file called "msdrpt.TXT".

none.
