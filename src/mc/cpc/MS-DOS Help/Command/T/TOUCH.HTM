TOUCH

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Change file timestamps.

This is a POSIX utility.

Syntax

TOUCH [/t YYYY MM DD HH NN SS] files

Parameters
files (NT4)
File or files to change date/timestamp.
Switches
/t YYYY MM DD HH NN SS (NT4)
Use the optional argument /t to specify a date other than the current time (four-digit years, two-digit months, days, hours, minutes, seconds).
Related

COPY - Copy one or more files to another location.
Equivalent Linux BASH commands:
touch - Change file timestamps.

Notes

none.

Examples

To set the date to 7:30 am 1st October 2015:

    TOUCH /t 2015 10 01 07 30 00 MyFile.txt

none.
