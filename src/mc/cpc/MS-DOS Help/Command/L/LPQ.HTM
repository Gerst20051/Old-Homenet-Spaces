LPQ

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

This diagnostic utility is used to obtain status of a print queue on a computer running the LPD server.

Syntax

LPQ /?

LPQ -SServer -PPrinter [-L]

Parameters

none.

Switches
/? (NTXP)
Displays help.
-L (NT3.5)
Specifies that a detailed status should be given.
-PPrinter (NT3.5)
Specifies the name of the printer for the desired queue.
-SServer (NT3.5)
Specifies the name of the computer that has the printer attached to it.
