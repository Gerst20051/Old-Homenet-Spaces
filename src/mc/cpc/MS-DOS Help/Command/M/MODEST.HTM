MODE (Set Typematic Rate)

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Sets the keyboard typematic rate, the rate at which the operating system repeats a character when you hold down the key for that character.

The typematic rate has two components, the rate and the delay. Some keyboards do not recognize this command.

Syntax

MODE /?

MODE CON[:] [RATE=r DELAY=d]

Parameters
CON[:] (v4.0 Win95 NT3.1)
Refers to the keyboard.
DELAY=d (v4.0 Win95 NT3.1)
Specifies the amount of time that must elapse - after you press and hold down a key - before the operating system starts to repeat the character. If you set the delay, you must also set the rate.
  • 1 0.25 seconds
  • 2 0.50 seconds (default)
  • 3 0.75 seconds
  • 4 1.00 seconds
RATE=r (v4.0 Win95 NT3.1)
Specifies the rate at which a character is repeated on the screen when you hold down a key. Valid values are in the range 1 through 32. These values are equal to approximately 2 to 30 characters per second, respectively. If you set the rate, you must also set the delay.
  • 1 2.0 CHAR/SECOND
  • 2 2.1 CHAR/SECOND
  • 3 2.3 CHAR/SECOND
  • 4 2.5 CHAR/SECOND
  • 5 2.7 CHAR/SECOND
  • 6 3.0 CHAR/SECOND
  • 7 3.3 CHAR/SECOND
  • 8 3.7 CHAR/SECOND
  • 9 4.0 CHAR/SECOND
  • 10 4.3 CHAR/SECOND
  • 11 4.6 CHAR/SECOND
  • 12 5.0 CHAR/SECOND
  • 13 5.5 CHAR/SECOND
  • 14 6.0 CHAR/SECOND
  • 15 6.7 CHAR/SECOND
  • 16 7.5 CHAR/SECOND
  • 17 8.0 CHAR/SECOND
  • 18 8.6 CHAR/SECOND
  • 19 9.2 CHAR/SECOND
  • 20 10.0 CHAR/SECOND (default IBM AT-compatible keyboards)
  • 21 10.9 CHAR/SECOND (default IBM PS/2-compatible keyboards)
  • 22 12.0 CHAR/SECOND
  • 23 13.3 CHAR/SECOND
  • 24 15.0 CHAR/SECOND
  • 25 16.0 CHAR/SECOND
  • 26 17.1 CHAR/SECOND
  • 27 18.5 CHAR/SECOND
  • 28 20.0 CHAR/SECOND
  • 29 21.8 CHAR/SECOND
  • 30 24.0 CHAR/SECOND
  • 31 26.7 CHAR/SECOND
  • 32 30.0 CHAR/SECOND
