  1. Successer to Windows XP.
  2. MINUMUM SYSTEM REQUIREMENTS (Vista Capable PC)
    1. http://www.microsoft.com/windowsvista/getready/capable.mspx
    2. CPU: 800MHz
    3. RAM: 512MB (may limit preformance and some features)
    4. HARD DRIVE SPACE: 15GB
    5. A graphics processor that is DirectX 9 capable.
  3. Windows Vista Premium Ready PC
    1. CPU: 1 GHz 32-bit (x86) or 64-bit (x64) processor
    2. RAM: 1 GB
    3. HARD DRIVE SPACE: 40 GB of hard drive capacity with 15 GB free space.
    4. Support for DirectX 9 graphics with a WDDM driver, 128 MB of graphics memory (minimum)2, Pixel Shader 2.0 and 32 bits per pixel.
    5. DVD-ROM Drive
    6. Audio output capability.
    7. Internet access capability.
  4. Windows Vista Editions:
    1. Business (Full Retail Price: $299 Upgrade Retail Price: $199) For Small and Medium Businesses - all the tools and features you need for running Vista in your business minus the features that are designed more for the home environment.
    2. Enterprise (Microsoft Enterprise "Software Assurance" Agreement ONLY) Windows Vista Enterprise is for the Enterprise. Small to Medium Businesses should stick to Windows Vista Busienss.
    3. Home Premium (Full Retail Price: $239 Upgrade Retail Price: $159) Windows Vista Home Premium is designed to give the home user the best Windows Vista experience - leaving out the more IT Pro focused features and tools.
    4. Home Basic (Full Retail Price: $199 Upgrade Retail Price: $99) Windows Vista Home Basic is designed for the home user’s very basic needs such as simply browsing the internet or checking email.
    5. Ultimate (Full Retail Price: $399 Upgrade Retail Price: $259) This is the "ultimate" that includes pretty much all the best features from all the other editions. For the uber Windows geek.
