W32TM

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

A tool used to diagnose problems occuring with Windows Time.

Syntax

W32TM /?

W32TM {/CONFIG [/COMPUTER:\\computer_name] [[/UPDATE] [/MANUALPEERLIST:list_of_computer_names]] [/SYNCFROMFLAGS:list_of_flags] | /MONITOR | /NTTE | /NTPTE | /REGISTER | /RESYNC [{\\computer_name] [/NOWAIT] | /REDISCOVER}] | /TZ | /UNREGISTER}

Parameters

none

Switches
/? (NTXP)
Displays help.
/COMPUTER:\\computer_name (NTXP)
\\computer_name (NTXP)
If ommitted, default to local computer.
/CONFIG (NTXP)
Adjusts time settings.
/MANUALPEERLIST:list_of_computer_names (NTXP)
Set time synchronization peers.
/MONITOR (NTXP)
Monitors the target computer or list of computers.
/NOWAIT (NTXP)
Do not wait for resynchronization. Forces redetection of network resourced before resynchronization.
/NTTE (NTXP)
Converts an NT system time into a readable format.
/NTPTE (NTXP)
Converts an NTP time into a readable format.
/REDISCOVER (NTXP)
Do not use current time resources.
/REGISTER (NTXP)
Register to run as a service and add default configuration to the registry.
/RESYNC (NTXP)
Resynchronize the clock as soon as possible, disregarding all accumulated error statistics.
/SYNCFROMFLAGS:list_of_flags (NTXP)
Set the source type used for synchronization, to:
  • MANUAL use the manual peer list
  • DOMHIER synchronize from a domain controller
/TZ (NTXP)
Display the current time zone settings.
/UNREGISTER (NTXP)
Unregister service and remove all configuration information from the registry.
/UPDATE (NTXP)
Configuration changes are used by Windows Time without the service being restarted.
