Computer Animation

Basic Computer Animation:

Make this a new and challenging hobby! Young people, who are interested in the basics of 3-D computer animation, can explore traditional and digital animation techniques. Create a short animation using various software packages.

Computer Animation:

This course is designed to enable the student to learn the basics of three dimensional computer animation and explore traditional and digital techniques of animation. All lessons are hands-on and taught by an experienced instructor in a computer lab. Students will produce a short animation using different software packages and present his/her animation to the class.

Computer Aided Design Technology:

CCBC Catonsville is a leader in Computer-Aided Design (CAD) in Maryland. This introductory course will provide you with a basic knowledge of CAD software used to create architectural and engineering drawings and to generate hard copy output.
