PERMS

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Display a user's ACL access permissions for a file.

Syntax

PERMS [account] [path] [/i] [/s]

Parameters
account (NT4)
User name.
path (NT4)
name of a file or folder in any legal format including UNC names Wildcards (* and/or ?) are permitted.
Switches
/i (NT4)
interactively logged on to the computer where the path resides (rather than being connected via the network).
/s (NT4)
Include subfolders.
Related

CACLS - Display or modify Access Control Lists (ACLs) for files and folders.

SHOWACL - Show file Access Control Lists NT2000.
SUBINACL - Change an ACL's user/domain (use when the file owner has moved to a new domain).
ATTRIB - Display or change file attributes.
XCACLS - Display or modify Access Control Lists (ACLs) for files and folders.
Equivalent Linux BASH commands:
chmod - Change access permissions.
chown - Change file owner and group.
