NET STATISTICS

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays the statistics log for the local Workstation or Server service.

Syntax

NET HELP STATISTICS

NET STATISTICS [{WORKSTATION | SERVER}]

Displays the services for which statistics are available:

NET STATISTICS

Parameters
SERVER (NT3.1)
Displays the Server service statistics.
WORKSTATION (NT3.1)
Displays the Workstation service statistics.
