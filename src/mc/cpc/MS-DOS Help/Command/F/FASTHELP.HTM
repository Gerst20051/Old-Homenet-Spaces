FASTHELP

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Displays a list of all commands and gives a brief explanation of each. The information that the FASTHELP command displays is similar to, but less detailed than, the information found in each individual commands' help.

Syntax

FASTHELP [command]

[command] /?

Parameters
command (v6.0)
Specifies the name of the command about which you want information. If you do not specify a command name, the FASTHELP command lists and briefly describes every command provided with the operating system.
Switches

none.

Related

For information about the operating system Help, see the HELP command.

Notes
  • Two ways to get online Help
Two ways to get online Help

You can use FASTHELP to get online Help for a command in two ways. You can specify the name of the command on the FASTHELP command-line, or you can type the name of the command and the /? switch at the command prompt. For example, you can type either of the commands to get information about the XCOPY command:

    FASTHELP XCOPY

or slightly faster:

    XCOPY /?
