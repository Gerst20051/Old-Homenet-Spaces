DEVINFO

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Prepares a device to use code pages. Place command in your CONFIG.SYS file.

Syntax

DEVINFO=devtype,subtype,[drive:][path]filename[,ROM=[,xxx[,yyy]

Parameters
devtype (NT3.1)
Specifies the type of device:
  • KBD keyboard
  • SCR monitor
  • parallel printer
[drive:][path]filename (NT3.1)
Specifies the file that contains information about the code pages for that device:
  • KEYBOARD.DCP keyboard
  • VIOTBL.DCP monitor
  • parallel printer
ROM= (NT3.1)
Unused, only for compatibility.
subtype (NT3.1)
Specifies the style or model of the device:
  • keyboard layout code (ie. FR120)
  • EGA monitor
  • VGA monitor
  • parallel printer
xxx (NT3.1)
Unused, only for compatibility.
yyy (NT3.1)
Unused, only for compatibility.
