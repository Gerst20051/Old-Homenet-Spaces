DEBUG

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

A program testing and editing tool.

After Debug starts, type ? to display a list of debugging commands.

Syntax

DEBUG [[drive:][path]filename [testfile-parameters]]

Parameters
( [drive:][path]filename v1.0 Win95 NT3.1)
Specifies the file you want to test.
testfile-parameters (v1.0 Win95 NT3.1)
Specifies command-line information required by the file you want to test.

none
