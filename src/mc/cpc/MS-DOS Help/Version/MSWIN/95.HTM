WINDOWS 95

Notes | Added | Changed | Removed | Files
  1. MINUMUM SYSTEM REQUIREMENTS
    1. http://support.microsoft.com/default.aspx?scid=kb;en-us;Q138349
    2. CPU: 386DX
    3. RAM: 4MB
    4. HARD DRIVE SPACE: 50-55MB
  2. Microsoft Windows 95 (Windows 4.0) is the official name (code-name "Chicago"), released August 24, 1995.
  3. Microsoft Windows 95 OEM Service Release (OSR) (code-name "Nashville"), released in 1995.
  4. Microsoft Windows 95 OEM Service Release (OSR) 1.0 Includes IE 2.0, released in 1996.
  5. Microsoft Windows 95 SP1 (95A Upgrade), released in February 1996.
  6. Microsoft Windows 95 OSR2, released in September 1996.
  7. Fully integrated 32-bit operating system replacing Windows 3.11, Windows for Workgroups 3.11, and MS-DOS as the mainstream desktop operating system.
  8. Windows 95 uses a version of MS-DOS to 'Bootup' (MS-DOS 7.0).
  9. Long Filename Support replaces 8dot3.
  10. Other major improvments over Windows 3.x (Still uses original FAT16).
  11. Able to display animated graphics while DOS loading.
  12. DriveSpace.
  13. Supports two new partition types (0E and 0F) to support the logical block addressing (LBA) INT13h extensions as specified in the Windows 95 Driver Development Kit (DDK).
  14. Internet Explorer 1.0.
  15. Upgrade version available (Win95UG).
  16. Educational Pack: Combined Win95 and NT4.0 for Teachers/Students only.
  17. Plus Pack: DriveSpace3, MS Themes, and 3D Pinball game.
  18. Windows 95B OSR 2.0 (Mainly a Floppy Disk Release), supports two new partition types (0B and 0C) to support the FAT32 file system. Maximum partition size now 2 TeraBytes (TB) But still 2.1 GBs if is FAT16 selected, released in 1996.
  19. Windows 95B OSR 2.1, USB support (CD-ROM version only), IE 3.0, released in 1996.
  20. Windows 95C OSR 2.5 with Integrated Internet Explorer 4.0, Improved USB Support, and some 95B/C issues fixed. Install CD-ROMs were 'Bootable' Dependent on BIOS Support, released in 1997.
