Mastering The Keyboard

Basic Skills:

It is never to soon to master keyboarding skills. Whether you want to work with word processing or computer graphics, you can learn correct keyboarding techniques and avoid the "hunt and peck" method as you have fun with Microsoft Word.

Master Skills:

This course is designed to enable the student to master computer keyboarding. Topics to be covered include introduction to the computer keyboard, correct finger placement, word processing, creating documents, and completing class projects.
