LOCK

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Locks a drive, enabling direct disk access for an application.

Syntax

LOCK [drive:]

Parameters
drive: (v7.00 Win95)
Specifies the drive to lock.
Switches

none.

Related

Can be used to lock a disk drive for direct disk access when using commands like: UNDELETE
To unlock a disk drive for direct disk access use: UNLOCK

Notes
  • Protect long filenames
Protect long filenames

It is necessary to LOCK the disk drive letter PRIOR to using commands like UNDELETE or you will have to reboot your computer with CTRL+ALT+DEL when you receive:

    "Windows has disabled direct disk access to protect your
    long filenames"

none.
