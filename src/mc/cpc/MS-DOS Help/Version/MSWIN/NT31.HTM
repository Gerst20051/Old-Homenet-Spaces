WINDOWS New Technology/N-Ten (NT) v3.1

Notes | Added | Changed | Removed | Files
  1. First BETA Testing of Windows NT, begins October 12 1992.
  2. Windows NT 3.1 beta 2 begins March 8 1993.
  3. Windows NT 3.1 beta 3 begins June 9 1993.
  4. Windows NT 3.1 released to manufacturing (RTM) July 26 1993.
  5. Windows NT 3.1 released (3.5" disks) August 1993.
  6. Windows NT 3.1 service pack (SP) 1 released October 1993.
  7. Microsofts first true 32bit Operating System.
  8. 6 million lines of code.
  9. Designed for Business & Networking Use.
  10. Not dependent on MS-DOS, uses its own NT FileSystem (NTFS) Bootstrap to install.
  11. Updates and Patches are released as "Service Packs", last known for 3.1 was SP-3 (October 1994).
  12. Originally targeted at the Intel i860, a RISC processor, which was behind schedule. Microsoft had to use an i860 emulator called the N10 (N-Ten).
