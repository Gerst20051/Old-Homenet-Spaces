Computer Vocabulary Quiz 2

An online self-grading quiz to help students prepare for the state-wide North Carolina Computer Skills Test given in the fall to 8th Graders. Written by parent Lee Ryan, former Blowing Rock School Webmaster and retired software engineer. September 2001.

Directions: select the best computer vocabulary term for each computer definition or description below. Check your work by clicking on the "Well...How did I do?" button at the end of the quiz.

1.  Key on a keyboard that is used (with letter keys) to key all capital letters.

2.  The process of loading the operating system software into memory to start up a computer. Also called "starting".

3. Style of font that slants the letters evenly to the right.

4.  Words in a piece of writing, file or document.

5.  A process using a computer to input and edit text; a computer application that resembles typewriting but allows instant correction of errors, moving text to different locations, and other editing functions.

6.  Key on a keyboard that is used to capitalize letters when pressed with that letter key.

7.  When keying text, the act of spacing text two lines apart.

8.  Special key on a keyboard that is used to for special purposes such as exiting a program. This special key is located on the same row as the function keys (F1, F2, etc.).

9.  A general term for pieces of information that a computer processes.

10.  A collection of data organized for search and retrieval.

11.  Sending a disk file from your computer to another computer.

12.  An input device resembling a typewriter and consisting of a standardized layout of buttons or keys with symbols, such as letters or numbers, that can be entered into a computer by pressing on the keys.

13.  Keys on a keyboard which contain the letters of the alphabet.

14.  Key on a keyboard that is used to enter information into a computer or to return the cursor to the beginning of a new line.

15.  A type of printer that produces high quality images using a method similar to that of a photo copying machine. While deskjet printers now produce high quality images, this type of printer is more advanced than a deskjet printer.

16.  The act of signing off of and disconnecting from a computer system.

17.  The act of connecting with a computer system and entering your identification and password.

18.  1. The starting point for your hands when beginning to keyboard using common keying techniques. 2. Keys on a keyboard which contain the letters a, s, d, f, j, k, l, ; and used for positioning the hands in typing.

19.  The device that reads from and writes to a floppy disk or hard disk.

20.  Law granting a legal right to a copyright holder which requires the permission of the copyright holder to make non-archival copies of the work in question.

21.  Key on a keyboard that moves the cursor to the left one space at a time, and on some keyboards and with most word processing software, deletes characters.

22.  A collection of related records in a database. 2. A word processing  or some other type of document, graphic, multimedia, etc. that is stored on a computer or external storage device with it's own name.

23.  To enter a program or file into a computer's memory.

24.  A device that permits a computer to transmit and receive data over a telephone line.

25.  Key on a keyboard that contains a special mark such as a tilda (~) or question mark (?).
