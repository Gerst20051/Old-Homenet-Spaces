IISRESET

Description | Syntax | Parameters | Switches | Related | Notes | Examples | Errorlevels | Availability

Restart Web services without rebooting the computer.

Syntax

IISRESET /?

IISRESET [computer_name] [/disable | /enable] [/noforce] [/reboot | /restart | /start | /status | /stop] [/rebootonerror] [/timeout:value]

Parameters
computer_name (NT2003)
Specifies the name of the computer to reset.
Switches
/? (NT2003)
Display help.
/disable (NT2003)
Disable restarting of Internet Services on the local system.
/enable (NT2003)
Enable restarting of Internet Services on the local system.
/noforce (NT2003)
Do not forcefully terminate Internet services if attempting to stop them gracefully fails.
/reboot (NT2003)
Reboot the computer.
/rebootonerror (NT2003)
Reboot the computer if an error occurs when starting, stoping, or restarting Internet services.
/restart (NT2003)
Stop and then restart all Internet services.
/start (NT2003)
Start all Internet services.
/status (NT2003)
Display the status of all Internet services.
/stop (NT2003)
Stop all Internet services.
/timeout:value (NT2003)
Specify the timeout value (in seconds) to wait for a successful stop of Internet services. On expiration of this timeout the computer can be rebooted if /REBOOTONERROR is specified. The default value is 20 for restart, 60 for stop, and 0 for reboot.
Related

none.

Notes

Introduced with IIS 5.0.

Only use to recover from serious problems that a Web server experiences. Changes made to the metabase may not be saved to disk losing any recent changes you made.
